# pwnscan

<p align="center">A modular framework designed to automate security tests.</p>

## Features 

* Simple Nmap Scan
* Advanced Nmap Scan
* Who-is Lookup
* DNS Lookup
* Reverse DNS Lookup

## Requirements: 

`go get github.com/fatih/color`
`go get github.com/likexian/whois-go`

## Usage: 

`go run main.go --url google.com`
`go run main.go -url google.com`
